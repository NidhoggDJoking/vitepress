基本命令
1. 插入模式
按 i 键进入插入模式，可以开始编辑文本。
2. 保存并退出
按 Esc 键退出插入模式，然后输入 :wq 保存并退出。
3. 仅保存
按 Esc 键退出插入模式，然后输入 :w 仅保存文件。
4. 仅退出
按 Esc 键退出插入模式，然后输入 :q 仅退出。如果文件有修改但未保存，使用 :q! 强制退出。
